SWITCH CASE

Q1.
WAP to print name of the month by accepting it's number from the user.

Input: Number (eg. 5)
Output: Respective month (May)

Q2.
WAP to print the weekdays according to it's sequence.

Input: Number (eg. 1)
Output: Weekday (eg. Monday)

Q3.
WAP to print single digit numbers in words.

Input: Number (eg. 1)
Output: Number in words (eg. one)

Q4.
Write a menu driven program to find out areas of different shapes like circle, triangle, rectangle and square.

Input: Required dimension of that particular shape.
Output: Area of that shape.

Q5.
Write a program to use functions of math.h library such as ceil, floor, squareroot, logarithm, absolute value using switch case.

Input: 1 number.
Output: value of respective function.

Q6.
Write a Menu-driven program to perform different arithmetic operations.

Input: 2 numbers and respective number for each arithmetic operations.
Output: Answer after performing that operation
