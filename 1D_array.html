1D ARRAY

Q1.
WAP to accept and print elements of a 1D array.

Input: size, elements.
Output: Array.

Q2.
WAP to delete an element from a 1D array.

Input: size, elements, element to be deleted.
Output: given array, array after deletion.

Q3.
WAP to find the maximum and minimum elements of a given 1D array.

Input: size, elements.
Output: given array, maximum and minimum element.

Q4.
WAP to find sum of all elements of a 1D array.

Input: size, elements.
Output: sum.

Q5.
WAP to sort the elements of a 1D array in ascending order.

Input: size, elements.
Output: sorted array.

Q6.
WAP to accept and merge 4 1D arrays.

Input: size, elements of 4 arrays.
Output: 4 arrays, merged array.

Q7.
WAP to reverse the elements of a 1D array.

Input: size, elements.
Output: Given array, reversed array.
