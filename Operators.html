OPERATORS

Q1.
Write a program to interchange the value of the two variables without using third variable.

Input: 2 numbers (for eg. x=5, y=7)
Output: Exchanged numbers (x=7, y=5)

Q2.
Write a program to rotate the content of three variables without using fourth variable

Input: 3 numbers (for eg. x=3, y=4, z=5)
Output: Rotated numbers (x=5, y=3, z=4)

Q3.
Write a program to exchange the digits of a two digit number

Input: number (for eg. x=35)
Output: reversed number (x=53)

Q4.
Write a program to accept a 4 digit number and display sum of digits and reverse of a number.

Input: 4-digit number (for eg. x=1234)
Output: Reversed number (x=4321)

Q5.
Write a program to accept a 4 digit number and to check whether it is perfect square number or not

Input: 4-digit number
Output: Print whether it's perfect square or not

Q6.
Write a program to display absolute value of an entered number using conditional operator.

Input: number
Output: Absolute value of number

Q7.
Write a program to find maximum of three numbers using conditional operator.

Input: 3 numbers
Output: Maximum of the 3 numbers
