FUNCTIONS

Q1.
WAP using functions to calculate a cube of a number, such that the main function only has function calling.

Input: 1 number.
Output: It's cube.

Q2.
WAP to find area of rectangle.

Input: Length, breadth.
Output: Area.

Q3.
WAP to find average of any given numbers.

Input: Number of numbers, individual nos.
Output: Average.

Q4.
WAP to find the number of ways of choosing r objects out of n distinct objects. (nCr)

Input: n,r.
Output: nCr.

Q5.
WAP to find the number of ways of arranging r objects out of n distinct objects. (nPr)

Input: n,r.
Output: nPr.

Q6.
WAP to print n elements of the Fibonacci series.

Input: n.
Output: n elements.

Q7.
Find sum of series: 1^1+2^2+3^3+4^4+.......n^n.

Input: n.
Output: sum.

Q8.
Find sum of series: x/2!-x/4!+x/6!-x/8!+........x/n!.

Input: n.
Output: sum.
